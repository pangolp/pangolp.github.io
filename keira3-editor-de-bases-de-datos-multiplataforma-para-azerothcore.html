Keira3

Aplicación de escritorio multiplataforma que incluye un Editor de bases de datos para el AzerothCore.

Con Keira3 no tienes que conocer el lenguaje SQL para cambiar los contenidos: generará las consultas SQL automáticamente por ti. El código SQL se mostrará, para que puedas aprender, y luego podrás copiarlo o ejecutarlo directamente en tu base de datos.

Hecho con ❤ y TypeScript, Electron, Angular, Bootstrap.

Apóyanos

Puedes apoyar el desarrollo de Keira3, realizando donaciones en PayPal.

Inspirado por

Creamos Keira inspirándonos en el antiguo editor de bases de datos indomit's Quice/Truice y en el Discover-'s SAI Editor. Nuestro objetivo principal era proporcionar un editor con las mismas características que fuera multiplataforma, así que lo construimos como una aplicación web.

Keira3 es el sucesor directo de Keira2. Mantuvimos la promesa de multiplicidad de plataformas, además de añadir la posibilidad de ejecutar las consultas generadas y simplificar la configuración de la aplicación.

Base de datos AzerothCore

Para utilizar Keira3 tienes que conectarte a una instancia de la base de datos de AzerothCore

Si no tienes ninguno, puedes crear uno fácilmente siguiendo esta guía.

Como alternativa, puedes utilizar una base de datos pública de AzerothCore con acceso de sólo lectura. Detalles aquí.

Cómo instalar Keira3

Para usar Keira3, no necesitas instalar ninguna dependencia. Basta con descargar y ejecutarlo.

Si utiliza Arch Linux, puede encontrar el paquete en AUR

Si está utilizando una base de datos MySQL 8.0 puede encontrarse con este error:

Client does not support authentication protocol requested by server; consider upgrading MySQL client

Antes de registrarte con Keira recuerda ejecutar esta consulta en tu base de datos para los usuarios que desees:

ALTER USER 'acore' IDENTIFIED WITH mysql_native_password BY 'acore';
flush privileges;
